START WITH STANDARDS

If you don't have full measurements, you might want to try using the CYCA standard body measurements based on your full bust/chest.

Essential measurements can be used to create straight-sided sweaters with long sleeves. Other sweaters will require extra measurements. We'll ask for these as we need them, but we recommend that you add them now if you think you'll need them later (for waist shaping, for example, or other sleeve/hem lengths).

Body circumferences should be your body's actual measurements at the points indicated. All lengths should be taken based on your preferences, as CustomFit will use them directly. Please enter all measurements in decimal form.

All Essential measurements are required to save the measurement set. After saving, you can add missing measurements but you won't be able to change existing numbers.
