For most women, I have seen that when sufficient vertical darts are added to the sweater, horizontal (or short row) darts become unnecessary. (This is especially true if the fit is "average" or "close", since the nature of these fits will keep the garment properly anchored on your body.)

That said, some knitters feel differently and are much more comfortable with both vertical and horizontal bust darts. For proper placement, start working the dart a couple of inches below the fullest part of your bust. This will ensure wrapped stitches stay on the under-side of the bust, where they are less noticeable.

If you’d like to add horizontal / short row bust darts to your sweater, here’s how:

{% if piece.is_cardigan %}

For Right Front:

Place one marker approximately 1" / 2.5 cm in from the armhole edge of your knitting.

Dart set-up row one (RS): Work to marker, wrap and turn.

Dart set-up row two (WS): Work even to neck edge.

    {% for height_in_inches, stitches_between_wraps, unused, half_num_wraps in piece.bust_dart_params %}

  • To work {{ height_in_inches | length_fmt }} of short row dart:

    Dart row one (RS): Work to {{ stitches_between_wraps | count_fmt }} stitches before the last wrapped stitch, wrap and turn.

    Dart row two (WS): Work even to neck edge.

    Repeat Dart rows one and two until a total of {{ half_num_wraps | count_fmt }} wraps have been worked; {{ half_num_wraps | count_fmt }} wraps will be worked on other cardigan piece, too.

    • {% endfor %}

For Left Front, repeat these instructions, except work short rows on WS rows, and work RS rows even.

{% else %}

When you’re ready to work the dart (typically ~2" / 5 cm below the fullest part of your bust), place two markers approximately 1" / 2.5 cm in from the outside edges of your knitting.

Dart set-up row one (RS): Work to last marker, wrap and turn.

Dart set-up row two (WS): Work to last marker, wrap and turn.

    {% for height_in_inches, stitches_between_wraps, height_in_rows, half_num_wraps in piece.bust_dart_params %}

  • To work {{ height_in_inches | length_fmt }} of short row dart:

    Dart row one (RS): Work to {{ stitches_between_wraps | count_fmt }} stitches before the last wrapped stitch, wrap and turn.

    Dart row two (WS): Work to {{ stitches_between_wraps | count_fmt }} stitches before the last wrapped stitch, wrap and turn.

    Repeat Dart rows one and two until a total of {{ height_in_rows | count_fmt }} wraps have been worked; {{ half_num_wraps | count_fmt }} wraps worked on each side.

    • {% endfor %}

All darts:

Close the dart as follows:

Dart closing row one (RS): Work to end of row, working all wraps with their stitches.

Dart closing row two (WS): Work to end of row, working all wraps with their stitches.

{% endif %}
