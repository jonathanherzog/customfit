Creating new designs

If you'd like to create a new prefilled design, you do that through the admin interface:

  • Add a new sweater design. Make sure you:
    • Add a picture.
    • Add a description
    • DO set the 'visibility' to 'Viewable by all CF users' (or subscribers only, if you like).
  • Your design is now available for viewing at /designs/personalize/(NUMBER), where (NUMBER) is the database ID of the design you just created. You can view this on the design admin page. Use the "By can be user visible only" filter to make it easier to find.
